I am a software developer based in Helsinki, Finland. This is a portfolio of some of my programming projects from school.

Doom Nukem

3D FPS engine made in C, no hardware acceleration. Features include:
  • Option to raytrace or bake shadows.
  • Raytraced mirrors and windows with normal maps.
  • Animated 3D enemies.
  • Screenspace effects like bloom and ssao.
  • Interactable doors.

Web-scraper

Scrapes and visualizes data from Etuovi using Geopy. Each house being sold is represented by pixels in the resulting image. In this example the price per square meter is used to color the pixel. Scraped data includes:
  • Price
  • Address
  • Floor area
  • Year of construcion
  • Seller name

Fractal explorer

Move around in a couple different fractals live. From very simple calculations emerge complex structured images.

E-commerce Website

A simple e-commerce website made with PHP in a weekend. Supports user registration, shopping carts, checkout, and store management for admins.
